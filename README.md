Leerink Partners — Transaction Intelligence Dashboard

An interactive analytics dashboard built from Leerink Partners' publicly available transaction data, covering 1,346 unique deals from 2009 to 2026 across 14 sectors and $245B in disclosed capital.

What this dashboard demonstrates

Real-time period comparisons — current month, QTD and YTD vs prior year
Deal velocity and capital trends across 17 years of transaction history
Client lifetime value analysis — relationship depth across product lines
Cross-sector intelligence — how deals flow from PIPEs to IPOs to M&A
KPI framework designed for executive decision-making at the CAO and CFO level
Data sources

All data sourced from publicly disclosed transactions on leerink.com. Deduplicated and cleaned using Python and Pandas.
